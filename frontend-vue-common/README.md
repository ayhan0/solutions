## Interview Case - Vue Frontend (works with .NET + Spring)

### Ports
- Vue: http://localhost:5173
- ASP.NET: http://localhost:5080
- Spring: http://localhost:8080

### Env
Create `.env`:
- VITE_API_DOTNET=http://localhost:5080
- VITE_API_SPRING=http://localhost:8080

### Install & Run
```bash
npm i
npm run dev
